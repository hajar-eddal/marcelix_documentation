# Security

If you believe you found a security issue related to [Marcelix], please report it privately.

## How To Report

Email:

- <a href="mailto:support@marcelix.com">support@marcelix.com</a>

Please include:

- a short description of the issue
- the affected page or flow
- steps to reproduce
- screenshots or logs if they help
- your contact information for follow-up

## Please Do Not

- post live exploit details publicly before the issue is reviewed
- include real customer secrets, payment credentials, or personal data you do not own
- run destructive tests against production accounts or infrastructure

## Scope

Relevant issues include:

- account access bugs
- authorization or data-exposure bugs
- payout or billing flow bugs
- prompt or private-content exposure
- media access-control issues

Best-effort review and response timing depends on severity and reproducibility.

[Marcelix]: https://www.marcelix.com
